# VikeServe - Complete Daily Needs Super App

VikeServe is a comprehensive mobile-first web application designed to serve as a one-stop solution for daily needs in Kenya and beyond. It combines marketplace functionality, service booking, community features, and emergency services in a single platform.

## 🌟 Features

### Core Features
- **Multi-service Marketplace**: Buy and sell goods and services
- **Service Booking**: Hire professionals for various services
- **Housing Listings**: Find rooms, apartments, and plots
- **Community Alerts**: Share and view community safety alerts
- **Education Hub**: Find teachers, internships, attachments, and training
- **Emergency Services**: Quick access to emergency contacts

### Authentication
- Email/Password login
- Google Sign-In
- Facebook Sign-In (optional)

### User Roles
- **General Users**: Browse and book services
- **Service Providers**: Offer services and manage bookings
- **Business Owners**: List products and manage marketplace
- **Students**: Access educational opportunities
- **Admins/Founders**: Platform management
